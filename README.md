# j2cli5

Command-line interface to Jinja2 for templating in shell scripts

## Installation

    $ pip install git+https://github.com/suzuki-navi/j2cli5

## Usage

    $ j2cli5 template.j2 para1.yml param2.yml

## Development

    $ pip install -e .
